= Profile Assist

== Building

**Default build:**
[source,bash]
----
mvn clean verify
----

**Run the application with Maven:**
[source,bash]
----
mvn spring-boot:run
----

**Build the application enforcing quality gates:**
----
mvn clean verify -PqualityGates
----

**Build the Asciidoc documentation:**
[source,bash]
----
mvn verify -PgenerateDocs
----
HTML is published in `target/docs`
